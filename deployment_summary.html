Quick Start Guide

Get your n8n AI Assistant deployed in minutes

1

Prepare Your Project

Clone the repository and install dependencies using pnpm:

git clone <your-repo-url>
cd n8n-ai-assistant
pnpm install
2

Set Up Your Database

Choose a managed database service (Railway, PlanetScale, Neon) and obtain your connection string. Create a `.env` file with your database URL and other configuration variables.

3

Deploy to Your Cloud Provider

Connect your GitHub repository to Railway or Vercel. Configure environment variables in your cloud provider's dashboard. Trigger a deployment.

4

Run Migrations & Seed Data

After deployment, run database migrations and seed initial data:

pnpm db:push
pnpm tsx server/seed.ts
5

Verify & Go Live

Access your deployed application URL, test all features, and monitor logs for any issues. Your n8n AI Assistant is now live!
